# MyApp

This is a React Native application built with Expo.

## Description

The application provides a simple user interface for logging in with an email and password using Firebase Authentication.

## Getting Started

### Prerequisites

- Node.js
- Expo CLI

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/myapp.git
   ```
2. Navigate to the project directory:
   ```bash
   cd myapp
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running the Application

You can run the application on different platforms using the following commands:

- To start the development server:
  ```bash
  npm start
  ```
  or
  ```bash
  expo start
  ```

- To run on Android:
  ```bash
  npm run android
  ```
  or
  ```bash
  expo start --android
  ```

- To run on iOS:
  ```bash
  npm run ios
  ```
  or
  ```bash
  expo start --ios
  ```

- To run on the web:
  ```bash
  npm run web
  ```
  or
  ```bash
  expo start --web
  ```
